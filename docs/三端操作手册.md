# 《小小封藏传人》三端操作手册

本手册适用于本地开发环境 `http://127.0.0.1:3002`，覆盖以下三个端：

- 客户端：`http://127.0.0.1:3002/?role=customer`
- NPC 端：`http://127.0.0.1:3002/?role=npc`
- 摄影师端：`http://127.0.0.1:3002/?role=photographer`

当前测试环境默认工作人员 PIN：

- NPC：`9527`
- 摄影师：`6688`

说明：

- 客户端通过家庭手机号进入，不需要工作人员 PIN。
- NPC 端和摄影师端都必须先输入“工作人员姓名 + PIN”才能进入后台。
- 本手册中的截图保存在同目录下的 `operation-manual-assets` 文件夹。

## 一、客户端操作手册

### 1. 客户进入方式

打开客户专属地址：

`http://127.0.0.1:3002/?role=customer`

如果该家庭已经续接过旧护照，系统可能直接回到地图页；地图页右上会显示 `重走激活仪式`，点击后可回到填写姓名和祠堂点香流程。

### 2. 进入地图与片头

首次进入或续接旧护照后，页面会出现 15 秒电影感片头。点击 `开启修行` 后进入正式开场。

![客户端地图与片头](./operation-manual-assets/customer-map.png)

操作说明：

- 点击 `开启修行`
- 等待片头播放完成
- 如果需要重新走填写与祠堂流程，点击右上角 `重走激活仪式`

### 3. 填写传人信息

返回激活页后，填写姓氏、孩子名、家训，并选择华服。

![客户端填写页](./operation-manual-assets/customer-activation.png)

操作说明：

- 核对 `Verified Family` 家庭信息
- 填写 `家族姓氏`
- 填写 `传人名讳`
- 选填 `封藏家训 / 家族箴言`
- 选择一套 `定制仪式华服`
- 点击 `唤醒护照 • 开启传人`

### 4. 祠堂点香仪式

点击激活后，进入祠堂仪式页。

![客户端祠堂仪式](./operation-manual-assets/customer-shrine.png)

操作说明：

- 点击 `点燃法向信香`
- 点击 `诚恳揖首祭拜`
- 等待祈福动画结束
- 点击 `启圣启智 · 携福进发`

完成后会正式激活护照，并进入时间地图。

### 5. 客户在现场如何配合 NPC / 摄影师

客户现场需要做两件事：

- 在 `传人印信` 页向 NPC 出示实时二维码，供 NPC 扫码授勋
- 在需要上传照片时，向摄影师出示客户专属摄影链接或护照码

注意事项：

- 二维码票据为短时、一次性使用
- 同一张票据被 NPC 消费后，客户需刷新页面重新生成新票据
- 只有当前关卡被 NPC 点亮后，下一关才会在地图中解锁

## 二、NPC 端操作手册

### 1. NPC 进入方式

打开 NPC 专属地址：

`http://127.0.0.1:3002/?role=npc`

输入：

- 工作人员姓名：例如 `01号 NPC` 或 `验收NPC`
- PIN：`9527`

通过门禁后进入评分控制台。

### 2. NPC 控制台总览

NPC 控制台集成了三块功能：家庭名单导入、客户扫码绑定、当前关卡授勋。

![NPC 控制台](./operation-manual-assets/npc-dashboard.png)

操作说明：

- 左上区域：导入 `Excel / CSV` 家庭名单
- 中间区域：粘贴客户二维码内容、扫码结果或直接输入 `XC-...` 护照号
- 下方区域：选择当前关卡并点亮星级

### 3. 导入家庭名单

导入规则：

- 支持最多 100 个家庭
- 推荐表头：`家庭名称 / 联系人 / 手机号 / 备注`
- 也兼容：`familyLabel / contactName / contactPhone / note`
- 同一手机号只能对应一个家庭
- 导入成功后会覆盖当前名单

操作步骤：

1. 点击 `选择 Excel/CSV 文件`
2. 上传现场名单
3. 如需标准模板，可点击：
4. `下载 UTF-8 模板 CSV`
5. `下载 Excel 模板 .xlsx`

系统会自动：

- 校验手机号格式
- 拦截重复手机号
- 尝试修复常见中文乱码
- 记录前台报到日志

### 4. 绑定客户

客户完成线下任务后，NPC 需要先锁定该孩子，再执行授勋。

绑定方式：

- 扫客户出示的专属二维码
- 粘贴扫码枪结果
- 粘贴二维码链接
- 直接输入客户护照号 `XC-...`

操作步骤：

1. 在输入框粘贴客户二维码内容或护照号
2. 点击 `核验`
3. 页面出现已锁定客户信息后，才可进行授勋

### 5. 点亮关卡勋章

绑定成功后：

1. 选择当前关卡
2. 根据现场完成质量点亮星级
3. 点击 `确认授勋，点亮当前关口勋章`

结果：

- 当前关卡状态变为 `NPC LIT`
- 客户端会收到勋章更新
- 下一关会在客户端地图中解锁
- 如果本次评为 `5 星`，客户端会触发 2.5 秒金色光柱特效

## 三、摄影师端操作手册

### 1. 摄影师进入方式

打开摄影师专属地址：

`http://127.0.0.1:3002/?role=photographer`

输入：

- 工作人员姓名：例如 `验收摄影师`
- PIN：`6688`

![摄影师门禁页](./operation-manual-assets/photographer-login-filled.png)

### 2. 摄影师后台总览

通过门禁后，进入现场照片上传控制台。

![摄影师控制台](./operation-manual-assets/photographer-dashboard.png)

主要功能：

- 绑定当前客户
- 上传现场照片
- 填写照片标题、点位和备注
- 将内容实时送入客户 H5 相片长卷

### 3. 绑定客户

摄影师可通过以下方式绑定：

- 打开客户专属上传链接
- 扫描客户摄影师二维码
- 粘贴二维码内容
- 直接输入护照号 `XC-...`

操作步骤：

1. 在绑定输入框里粘贴客户链接或护照号
2. 点击 `绑定客户`
3. 确认页面已显示当前客户信息

### 4. 上传照片

绑定客户后：

1. 点击 `上传照片`
2. 选择现场原图或成片
3. 填写上传标题
4. 填写拍摄点位
5. 选填摄影备注
6. 点击 `送入 H5 长卷`

上传后效果：

- 照片会写入当前客户的相片长卷
- 客户端切到 `相片长卷` 时可查看新增照片

## 四、现场推荐操作顺序

建议现场按以下顺序执行：

1. 前台在 NPC 端导入本场家庭名单
2. 家庭到场后，客户在手机端输入手机号完成核验
3. 客户首次进入时完成填写信息和祠堂仪式
4. 客户在每一关结束后向 NPC 出示实时二维码
5. NPC 扫码锁定客户并点亮当前关卡勋章
6. 摄影师按需绑定客户并上传现场照片
7. 客户回到手机端查看地图解锁、勋章状态与相片长卷

## 五、常见问题

### 1. 为什么客户一进去就是地图，没有填写页？

原因：

- 该家庭已经续接过旧护照

处理：

- 等待片头结束
- 点击右上角 `重走激活仪式`

### 2. 为什么 NPC 扫第二次二维码失败？

原因：

- 客户二维码票据是一次性的

处理：

- 让客户刷新 `传人印信` 页，生成新的二维码

### 3. 为什么导入名单后有乱码？

处理：

- 优先使用 `下载 Excel 模板 .xlsx`
- 或将原始 CSV 另存为 `UTF-8 CSV` 后重新导入

### 4. 为什么工作人员能直接进后台？

前提是设备上已有有效门禁会话。当前实现下，工作人员登录成功后，本设备授权默认有效 8 小时。
