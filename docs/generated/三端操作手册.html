《小小封藏传人》三端操作手册

适用于本地开发环境 http://127.0.0.1:3002，覆盖客户端、NPC 端、摄影师端三套现场操作流程。

三端入口

  • 客户端：http://127.0.0.1:3002/?role=customer
  • NPC 端：http://127.0.0.1:3002/?role=npc
  • 摄影师端：http://127.0.0.1:3002/?role=photographer

测试 PIN

  • NPC：9527
  • 摄影师：6688

一、客户端操作手册

1. 客户进入方式

打开客户专属地址 ?role=customer。如果该家庭已续接旧护照，系统可能直接回到地图页；片头结束后，点击右上角 重走激活仪式 可回到填写姓名和祠堂点香流程。

2. 重走激活仪式

如果客户不是首次进入，地图右上会出现 重走激活仪式。点击后回到激活页，重新填写孩子信息和家训。

3. 祠堂点香

激活页提交后会进入祠堂仪式页。客户需按顺序完成点香、祭拜与启智三个动作，完成后才正式激活护照。

4. 现场配合方式

  • 在 传人印信 页向 NPC 出示实时二维码，供 NPC 扫码授勋。
  • 在需要上传照片时，向摄影师出示客户专属上传链接或护照号。
  • 二维码票据为一次性使用；被 NPC 消费后，需要刷新印信页生成新票据。

二、NPC 端操作手册

1. NPC 登录

打开 ?role=npc，输入工作人员姓名和 PIN 9527。验证通过后进入评分控制台。

2. 导入家庭名单

  • 支持最多 100 个家庭。
  • 推荐表头：家庭名称 / 联系人 / 手机号 / 备注。
  • 也兼容：familyLabel / contactName / contactPhone / note。
  • 同一手机号只能对应一个家庭。
  • 导入成功后覆盖当前名单，并记录前台报到日志。

如需模板，可点击 下载 UTF-8 模板 CSV 或 下载 Excel 模板 .xlsx。

3. 绑定客户并授勋

  1. 扫描客户出示的专属二维码，或粘贴二维码内容 / 护照号 XC-...。
  2. 点击 核验，确认当前客户已被锁定。
  3. 选择当前关卡并点亮星级。
  4. 点击 确认授勋，点亮当前关口勋章。

授勋后，客户端会实时收到勋章更新，并解锁下一关。评为 5 星 时，客户端会触发 2.5 秒金色光柱特效。

三、摄影师端操作手册

1. 摄影师登录

打开 ?role=photographer，输入工作人员姓名和 PIN 6688。

2. 摄影师后台

后台支持绑定客户、上传照片、填写标题与点位，并将内容实时送入客户 H5 长卷。

3. 上传流程

  1. 通过客户专属上传链接、二维码内容或护照号 XC-... 绑定客户。
  2. 点击 上传照片 选择本地图片。
  3. 填写标题、拍摄点位，可选填摄影备注。
  4. 点击 送入 H5 长卷，照片会同步到当前家庭的相片长卷。

四、现场推荐顺序

  1. 前台在 NPC 端导入本场家庭名单。
  2. 家庭到场后，客户在手机端输入手机号完成核验。
  3. 客户首次进入时完成填写信息和祠堂仪式。
  4. 客户在每一关结束后向 NPC 出示实时二维码。
  5. NPC 扫码锁定客户并点亮当前关卡勋章。
  6. 摄影师按需绑定客户并上传现场照片。
  7. 客户回到手机端查看地图解锁、勋章状态与相片长卷。

五、常见问题

1. 为什么客户一进去就是地图？

该家庭已经续接过旧护照。等待片头结束后，点击右上角 重走激活仪式 即可回到填写页。

2. 为什么 NPC 扫第二次二维码失败？

客户二维码票据是一次性的。让客户刷新 传人印信 页，生成新的二维码后再次扫码。

3. 为什么名单导入后有乱码？

优先使用 Excel 模板，或将源 CSV 另存为 UTF-8 CSV 后重新导入。

4. 为什么工作人员能直接进后台？

前提是设备上已有有效门禁会话。当前实现下，工作人员登录成功后，本设备授权默认有效 8 小时。
